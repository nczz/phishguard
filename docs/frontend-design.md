# 前端設計文件

## UX 設計原則

1. **三步完成** — 客戶從登入到發出第一個釣魚測試，不超過 5 分鐘
2. **預設就對** — 所有設定都有合理預設值，客戶不需要理解每個參數
3. **報表即價值** — 漏斗圖、部門排名、趨勢圖是客戶最常看的畫面，要最醒目
4. **角色分流** — Platform Admin 和 Tenant 使用者看到完全不同的介面

## 技術選型

| 項目 | 選擇 | 理由 |
|------|------|------|
| UI 框架 | Ant Design 5 | Table/Form/Steps/Card 元件完整，適合後台 |
| 圖表 | Recharts | 輕量、React 原生、漏斗圖支援好 |
| 路由 | React Router 6 | 標準選擇 |
| HTTP | Axios | 攔截器方便處理 JWT 和錯誤 |
| 狀態 | React Context + hooks | 規模不需要 Redux |

- 主色：#1677ff（Ant Design 預設藍）

- 危險/高風險：#ff4d4f
